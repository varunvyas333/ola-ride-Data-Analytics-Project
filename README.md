# 🚖 Ola Booking Analysis Dashboard

## Project Overview

This project focuses on analyzing Ola ride booking data using Python (Pandas), MySQL, and Power BI. The goal of the project is to understand booking patterns, revenue generation, customer satisfaction, cancellation trends, and vehicle performance through data analysis and visualization.

The project follows a complete data analytics workflow starting from data cleaning and preprocessing, followed by SQL-based analysis and dashboard creation in Power BI.

---

## Objectives

- Analyze overall booking performance.
- Identify revenue generated by different vehicle types.
- Study customer and driver ratings.
- Analyze ride cancellation patterns.
- Compare vehicle performance based on bookings and revenue.
- Generate business insights using interactive dashboards.

---

## Project Workflow

1. Collected the Ola booking dataset.
2. Cleaned and preprocessed the data using Pandas.
3. Imported the cleaned dataset into MySQL.
4. Performed SQL analysis using multiple business queries.
5. Connected the data to Power BI.
6. Created interactive dashboards and visualizations.
7. Generated business insights from the results.

- Total Bookings: 20.41K

- Cancellation Rate: 38.00%
